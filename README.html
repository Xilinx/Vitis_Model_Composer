Rapid design exploration using Vitis Model Composer

Within the Simulink environment, Vitis Model Composer offers a range of performance-optimized blocks that facilitate the design and implementation of DSP algorithms on AMD devices. The inclusion of the Vitis Model Composer AI Engine, HLS, and HDL libraries enables efficient algorithm exploration and greatly expedites the path towards production.

To learn more, visit the Vitis Model Composer website.

What's new in 2023.1?

  • Take a look at the latest tutorial series on AI Engines. This set of tutorials provides an excellent introduction to AI Engines and guides you through the process of developing a DSP algorithm on AI Engines using Vitis Model Composer.
  • The new Channelizer example implements a high-speed channelizer design using a combination of AI Engine and Programmable Logic (PL) resources in Versal devices.
  • The filtering in frequency domain design showcases filtering in frequency domain in AI Engines and also shows how to increase the throughput using different techniques.
